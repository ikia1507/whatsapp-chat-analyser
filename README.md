WhatsApp Chat Analyzer

A Streamlit web app that analyzes WhatsApp chat exports and provides insights such as:
- Message counts
- Most active users
- Timelines
- Word usage
- Emoji analysis
- Activity patterns


 Features
- Upload WhatsApp `.txt` chat file
-->make sure while exporting chat from whatsapp select without media option<--
- Analyze individual users or overall chat
- Visualizations using Matplotlib & Seaborn
- Handles emojis and multilingual text
